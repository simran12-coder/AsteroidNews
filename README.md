# AsteroidNews

## Example project which uses SceneKit framework to show objects with Near-Earth Asteroids information.

## Frameworks
1. SceneKit
2. Alamofire
3. SwiftyJSON

## Features
- Uses SceneKit to show Start Particles and rotating Earth.
- Uses NeoWs Api to show asteroid information
